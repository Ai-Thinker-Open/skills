# WS2812 RGB LED Control API Reference

> Source files: `components/stage/ws2812/ws2812.h` (core)  
> `ws281x_spi.h` (SPI mode) / `ws281x_ir.h` (IR mode)  
> A library for controlling WS2812/NeoPixel LED strips, supporting two hardware driving methods: SPI and IR.

---

## Overview

WS2812 is a single-wire bus RGB LED. BL602 drives it in two ways:
- **SPI mode**: High-speed SPI output timing, low CPU load
- **IR mode**: Uses IR infrared module's PWM output timing (pin fixed to GPIO11)

The `ws2812` library on BL602 is based on a color mode abstraction layer, supporting HSV color space with global brightness control.

---

## Header Files

```c
#include "ws2812.h"

// SPI mode (requires defining WS281X_SPI_MODE)
#include "ws281x_spi.h"

// IR mode (requires defining WS281X_IR_MODE)
#include "ws281x_ir.h"
```

## Type Definitions

### `color_t`

Color structure (from `color_mode.h`):

```c
typedef struct {
    uint8_t r;   // Red component (0-255)
    uint8_t g;   // Green component (0-255)
    uint8_t b;   // Blue component (0-255)
} color_t;
```

### `ws2812_dev_t`

Single LED device descriptor:

```c
typedef struct {
    color_t color;       // Current color
    float *brightness;  // Pointer to global brightness (pointer enables sharing)
    uint8_t index;      // LED index
} ws2812_dev_t;
```

### `ws2812_strip_t`

Whole LED strip descriptor:

```c
typedef struct {
    ws2812_dev_t *dev;     // LED array
    uint8_t led_count;     // Number of LEDs
    float brightness;      // Global brightness (0.0-1.0)
    uint8_t pin;           // GPIO pin (used in SPI mode)
} ws2812_strip_t;
```

---

## Global Variables

```c
extern ws2812_strip_t *ws2812_strip_dev;
```

> After initialization, points to the internally allocated strip structure.

## Core Functions

### `ws2812_init`

Initializes the LED strip (automatically selects SPI or IR mode).

```c
void ws2812_init(ws2812_strip_t *ws2812_strip);
```

| Parameter | Description |
|-----------|-------------|
| `ws2812_strip` | LED strip configuration structure pointer, `led_count` and `pin` must be set |

> Internally selects the driver based on compile-time macros `WS281X_SPI_MODE` or `WS281X_IR_MODE`.

---

### `ws2812_set_all_pixels_color`

Sets all LEDs to the same color.

```c
void ws2812_set_all_pixels_color(uint8_t r, uint8_t g, uint8_t b, float brightness);
```

| Parameter | Description |
|-----------|-------------|
| `r/g/b` | RGB components (0-255) |
| `brightness` | Global brightness (0.0-1.0), set to 1.0 to not be affected by global `brightness` |

---

### `ws2812_show_leds`

Flushes color data to the LED strip (sends DMA/IR frame).

```c
void ws2812_show_leds(void);
```

> This function must be called after all `set_pixel_color` calls for changes to actually display.

---

### `ws2812_set_pixel_color`

Sets the color of a single LED (in memory, does not auto-refresh).

```c
void ws2812_set_pixel_color(uint8_t index, uint8_t r, uint8_t g, uint8_t b);
```

| Parameter | Description |
|-----------|-------------|
| `index` | LED position (starting from 0) |
| `r/g/b` | RGB components (0-255) |

---

### `ws2812_set_pixel_brightness`

Sets the brightness factor of a single LED (overrides global brightness).

```c
void ws2812_set_pixel_brightness(uint8_t index, float brightness);
```

| Parameter | Description |
|-----------|-------------|
| `index` | LED position |
| `brightness` | Brightness factor (0.0-1.0) |

---

### `ws2812_set_global_brightness`

Sets the global brightness for all LEDs (requires re-calling `show` after modification).

```c
void ws2812_set_global_brightness(float brightness);
```

| Parameter | Description |
|-----------|-------------|
| `brightness` | Global brightness (0.0-1.0) |

---

### `ws2812_set_pixel_color_hsv`

Sets a single LED's color (HSV color space, automatically applies brightness).

```c
void ws2812_set_pixel_color_hsv(uint8_t index, uint8_t h, uint8_t s, uint8_t v);
```

| Parameter | Description |
|-----------|-------------|
| `index` | LED position |
| `h` | Hue (0-255, corresponds to 0-360°) |
| `s` | Saturation (0-255) |
| `v` | Value/brightness (0-255) |

---

### `ws2812_set_led_count`

Modifies the LED count at runtime (will re-allocate memory).

```c
void ws2812_set_led_count(uint8_t count);
```

---

### `ws2812_get_led_count`

Gets the current LED count.

```c
uint8_t ws2812_get_led_count(void);
```

## Usage Examples

### SPI Mode Driving 8 WS2812 LEDs

```c
#include "ws2812.h"
#include "ws281x_spi.h"

// Define LED strip configuration
static ws2812_strip_t g_ws2812_strip = {
    .led_count = 8,
    .pin = 10,  // GPIO10 as SPI MOSI
    .brightness = 0.5,
};

void ws2812_spi_demo(void)
{
    // Use SPI mode
#ifdef WS281X_SPI_MODE
    ws281x_spi_init(&g_ws2812_strip);

    // Set LED 0 to red
    ws281x_spi_set_pixel_color(0, 255, 0, 0);

    // Set all LEDs to green
    ws2812_set_all_pixels_color(0, 255, 0, 1.0);

    // Refresh display
    ws281x_spi_show_leds();
#endif
}
```

### IR Mode Rainbow Breathing Light

```c
#include "ws2812.h"
#include "ws281x_ir.h"

static ws2812_strip_t g_strip = {
    .led_count = 8,
    .brightness = 0.8,
};

void rainbow_breathing(void)
{
#ifdef WS281X_IR_MODE
    ws281x_ir_init(&g_strip);

    while (1) {
        // HSV rainbow effect
        for (uint8_t h = 0; h < 255; h += 5) {
            for (uint8_t i = 0; i < g_strip.led_count; i++) {
                ws2812_set_pixel_color_hsv(i, h + i * 30, 255, 200);
            }
            ws281x_ir_show_leds();
            vTaskDelay(pdMS_TO_TICKS(30));
        }

        // Breathing effect
        for (float b = 0.0; b <= 1.0; b += 0.1) {
            ws2812_set_global_brightness(b);
            ws281x_ir_show_leds();
            vTaskDelay(pdMS_TO_TICKS(50));
        }
        for (float b = 1.0; b >= 0.0; b -= 0.1) {
            ws2812_set_global_brightness(b);
            ws281x_ir_show_leds();
            vTaskDelay(pdMS_TO_TICKS(50));
        }
    }
#endif
}
```

### Chasing LEDs (Marquee Effect)

```c
void chase_animation(void)
{
    ws2812_init(&g_ws2812_strip);
    ws2812_set_all_pixels_color(0, 0, 0, 1.0);

    while (1) {
        for (uint8_t i = 0; i < ws2812_get_led_count(); i++) {
            ws2812_set_pixel_color(i, 255, 255, 255);
            ws2812_show_leds();
            vTaskDelay(pdMS_TO_TICKS(100));
            ws2812_set_pixel_color(i, 0, 0, 0);  // Turn off
        }
    }
}
```
