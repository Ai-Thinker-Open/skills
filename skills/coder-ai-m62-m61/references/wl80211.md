# wl80211 Wi-Fi Driver Interface

## Overview

`wl80211` is the core Wi-Fi driver module for BL616/BL618, located at `components/wireless/wl80211/`, providing complete IEEE 802.11 wireless LAN driver support. It serves as the low-level foundation for the `wifi_mgmr` upper-level API, directly operating the MAC software layer (macsw) and WPA Supplicant.

## Global State

```c
extern struct wl80211_global_state wl80211_glb;
```

`wl80211_glb` contains the driver global state:

| Field | Type | Description |
|-------|------|-------------|
| `country` | `const struct ieee80211_dot_d *` | Country code configuration |
| `scanning` | bit | Scanning in progress |
| `connecting` | bit | Connecting in progress |
| `associated` | bit | Associated |
| `authenticating` | bit | Authenticating |
| `link_up` | bit | Connection established |
| `ap_en` | bit | AP mode enabled |
| `monitor_en` | bit | Monitor mode enabled |
| `connect_params` | `struct wl80211_connect_params *` | Current connection parameters |
| `aid` | `uint16_t` | Association ID |

## Initialization

```c
void wl80211_init(void);
```

Initializes the wl80211 driver. It is recommended to call through `wifi_mgmr_init()` rather than directly.

## Event Types

```c
enum {
    WL80211_EVT_SCAN_DONE,        // Scan complete
    WL80211_EVT_STA_CONNECTED,    // STA connection succeeded
    WL80211_EVT_STA_DISCONNECTED, // STA disconnected
    WL80211_EVT_AP_STARTED,       // AP started
    WL80211_EVT_AP_STOPPED,       // AP stopped
    WL80211_EVT_AP_STA_ADDED,     // STA joined under AP
    WL80211_EVT_AP_STA_DEL,       // STA left under AP
};
```

Events are asynchronously delivered via `wl80211_post_event()`:

```c
void wl80211_post_event(int code1, int code2);
```

## Control Commands

`wl80211_cntrl()` is the central interface for driver control:

```c
int wl80211_cntrl(int cmd, ...);
```

Command list:

| Command | Description | Parameters |
|---------|-------------|------------|
| `WL80211_CTRL_SCAN` | Start scan | `struct wl80211_scan_params *` |
| `WL80211_CTRL_STA_CONNECT` | STA connect | `struct wl80211_connect_params *` |
| `WL80211_CTRL_STA_DISCONNECT` | STA disconnect | None |
| `WL80211_CTRL_STA_SET_PS_MODE` | Set power mode | `int mode` |
| `WL80211_CTRL_STA_GET_MAC` | Get STA MAC | `uint8_t mac[6]` |
| `WL80211_CTRL_STA_GET_RSSI` | Get RSSI | `int *rssi` |
| `WL80211_CTRL_AP_START` | Start AP | `struct wl80211_ap_settings *` |
| `WL80211_CTRL_AP_STOP` | Stop AP | None |
| `WL80211_CTRL_AP_SET_SSID` | Set AP SSID | `uint8_t ssid[33]` |
| `WL80211_CTRL_AP_SET_PASSWORD` | Set AP password | `uint8_t password[65]` |
| `WL80211_CTRL_MONITOR_START` | Start monitor mode | `struct wl80211_monitor_settings *` |
| `WL80211_CTRL_MONITOR_STOP` | Stop monitor mode | None |
| `WL80211_CTRL_INJECT_FRAME` | Inject 802.11 frame | buffer, len |

## Convenience Inline Functions

### Scan

```c
static inline int wl80211_scan(struct wl80211_scan_params *param);
```

Returns: `0` success, `-1` country code not set, `-2` scanning not allowed in monitor mode.

### STA Connection

```c
static inline int wl80211_sta_connect(struct wl80211_connect_params *param);
```

Returns: `0` success, `-1` connection not allowed in monitor mode, `-2` connection already in progress, `-3` invalid SSID, `-4` invalid channel, `-5` invalid IE parameter.

## Working Code Examples

### STA Scan and Connection

```c
#include "wl80211.h"
#include "wifi_mgmr_ext.h"
#include <string.h>

static void wifi_event_cb(int code, int val, void *arg)
{
    switch (code) {
    case CODE_WIFI_ON_CONNECTED:
        printf("Wi-Fi connected\r\n");
        break;
    case CODE_WIFI_ON_DISCONNECT:
        printf("Wi-Fi disconnected\r\n");
        break;
    case CODE_WIFI_ON_SCAN_DONE:
        printf("Scan done, found %d APs\r\n", val);
        break;
    case CODE_WIFI_ON_GOT_IP:
        printf("Got IP\r\n");
        break;
    }
}

void wifi_sta_example(void)
{
    struct wl80211_scan_params scan_params = { 0 };
    struct wl80211_connect_params conn_params = { 0 };

    /* Set country code (required before scanning) */
    wl80211_cntrl(WL80211_CTRL_SET_COUNTRY, "CN");

    /* Start passive scan */
    scan_params.flags = WL80211_SCAN_FLAGS_PASSIVE;
    scan_params.probe_cnt = 3;
    scan_params.probe_interval = 100; /* TU */
    int ret = wl80211_scan(&scan_params);
    printf("scan ret=%d\r\n", ret);

    /* Connect to AP */
    memset(&conn_params, 0, sizeof(conn_params));
    strncpy((char *)conn_params.ssid, "MyAP", 32);
    strncpy((char *)conn_params.password, "password123", 64);
    conn_params.auth_type = WL80211_AUTHTYPE_OPEN_SYSTEM;
    conn_params.mfp = WL80211_MFP_NO;
    conn_params.channel = 6;

    ret = wl80211_sta_connect(&conn_params);
    printf("connect ret=%d\r\n", ret);

    /* Check connection status */
    if (wl80211_sta_is_connected()) {
        uint8_t bssid[6];
        int channel = wl80211_sta_get_channel();
        wl80211_sta_get_bssid(bssid);
        printf("Connected to %s, channel=%d\r\n", conn_params.ssid, channel);
    }
}
```

### Monitor Mode

```c
static void monitor_rx_cb(void *ctx, void *pkt, size_t len,
                          size_t mac_hdr_len, int rssi)
{
    printf("RX frame: len=%u, rssi=%d\r\n", (unsigned)len, rssi);
    /* Process raw 802.11 frame */
}

void wifi_monitor_example(void)
{
    struct wl80211_monitor_settings mon = {
        .channel_width = WL80211_CHAN_WIDTH_20,
        .center_freq1 = 2437, /* channel 6 */
        .recv_ctx = NULL,
        .recv = monitor_rx_cb,
    };

    wl80211_cntrl(WL80211_CTRL_MONITOR_START, &mon);
    printf("Monitor mode started on channel 6\r\n");

    /* ... receive processing ... */

    wl80211_cntrl(WL80211_CTRL_MONITOR_STOP);
}
```

### Low Power Mode

```c
void wifi_ps_example(void)
{
    /* Enter low power mode (PS-Poll) */
    wl80211_sta_set_ps(1);

    /* Exit low power mode */
    wl80211_sta_set_ps(0);
}
```

## Notes

1. **Country code must be set before scanning**, otherwise `wl80211_scan()` returns `-1`
2. **Monitor mode** and **STA/AP mode** are mutually exclusive; stop the current mode before switching
3. `wl80211_glb.link_up` is only set after successful association (receiving Assoc Resp)
4. WPA3 (SAE) connections require setting `auth_type = WL80211_AUTHTYPE_SAE`
5. It is recommended to use `wifi_mgmr_*` API rather than directly operating `wl80211_cntrl()`, unless advanced control is needed
