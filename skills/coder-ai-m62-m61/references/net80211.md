# net80211 Wi-Fi MLME Layer Documentation

**Reference:** `bouffalo_sdk/components/wireless/wl80211/`  
**Header:** `macsw/wl80211_mac.h`

## Overview

The net80211 MLME (MAC Sublayer Management Entity) layer in wl80211 handles 802.11 station management: scan, authentication, association, beacon monitoring, and connection state machine. It sits between the high-level `wifi_mgmr` API and the `macsw` (MAC Software) layer.

```
┌─────────────────────────────────────────────────────────────────┐
│                        wifi_mgmr API                            │
│                    (wifi_mgmr_sta_connect, etc.)                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                     wl80211 Driver API                          │
│                      (wl80211_cntrl)                            │
└─────────────────────────────────────────────────────────────────┘
                              │
          ┌───────────────────┴───────────────────┐
          ▼                                       ▼
┌─────────────────────┐               ┌─────────────────────────────┐
│   net80211 MLME     │               │      macsw Layer            │
│  (scan/connect ops)  │──────────────▶│  (wl80211_mac_do_scan,     │
│                     │               │   wl80211_mac_do_connect)   │
└─────────────────────┘               └─────────────────────────────┘
          │                                       │
          └───────────────────┬───────────────────┘
                              ▼
                    ┌─────────────────────┐
                    │  bl_wpa_supplicant  │
                    │  (4-way handshake) │
                    └─────────────────────┘
```

## 11. Complete Usage Example

```c
#include "wifi_mgmr.h"
#include "wl80211.h"

// Event handler for WiFi events
void wifi_event_handler(uint32_t event_code)
{
    switch (event_code) {
        case CODE_WIFI_ON_INIT_DONE:
            printf("WiFi initialized\n");
            break;
        case CODE_WIFI_ON_SCAN_DONE:
            printf("Scan complete\n");
            // Process scan results
            break;
        case CODE_WIFI_ON_CONNECTED:
            printf("Connected!\n");
            // Get connection info
            wifi_mgmr_connect_ind_stat_info_t info;
            wifi_mgmr_sta_connect_ind_stat_get(&info);
            printf("SSID: %s, AID: %d\n", info.ssid, info.aid);
            break;
        case CODE_WIFI_ON_DISCONNECT:
            printf("Disconnected\n");
            break;
        case CODE_WIFI_ON_GOT_IP:
            printf("IP address obtained\n");
            break;
    }
}

int main(void)
{
    // Initialize WiFi manager (registers event handlers)
    wifi_mgmr_init();
    
    // Set country code for channel availability
    wifi_mgmr_set_country_code("US");
    
    // Start a scan
    wifi_mgmr_scan_params_t scan_params = {0};
    scan_params.probe_cnt = 2;
    wifi_mgmr_sta_scan(&scan_params);
    
    // After scan completes, connect to an AP
    wifi_mgmr_sta_connect_params_t conn_params = {0};
    strncpy(conn_params.ssid, "MyNetwork", sizeof(conn_params.ssid));
    strncpy(conn_params.key, "password", sizeof(conn_params.key));
    conn_params.use_dhcp = 1;
    wifi_mgmr_sta_connect(&conn_params);
    
    // Enable power save after connection
    wifi_mgmr_sta_ps_enter();
    
    return 0;
}
```
